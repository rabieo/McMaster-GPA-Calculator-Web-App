## McMaster GPA Calculator


## What is it?
Web app that can be used to store and calculate McMaster student's 12 point academic GPA. Currently only made for desktop.
https://rabieo.github.io/Mcmaster-GPA-Calculator/

## Technology used
* JavaScript
* React.js
* Bootstrap
* HTML/CSS
